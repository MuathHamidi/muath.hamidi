Place your official logos in this folder with the following exact filenames to automatically replace the high-quality SVG fallback logos in the presentation:

1. udem_logo.png (Université de Montréal logo)
2. mila_logo.png (Mila - Quebec AI Institute logo)
3. ivado_logo.png (IVADO logo)

Supported formats: PNG (recommended), JPG, or SVG.
If the image file is not found, the presentation slide will gracefully fall back to a beautifully stylized SVG rendering of the logo.
